## Planned Features

* General
    - Json Web Token (Jwt)
* Authentication
    - Handle user authentication/consent
    - External authentication provider
* Authorization
    - Authorization Code Grant with PKCE
    - Client Credentials Grant
    - Refresh Token Flow
